Student Exam Performance Indicator

Welcome! This project is a Machine Learning-powered web application that predicts a student's Math Score based on their demographic and academic information.

  • Input student details such as gender, ethnicity, parental education, lunch type, test preparation, and reading/writing scores.
  • Get an instant prediction of the expected math score using advanced regression models.
  • Built with Flask, Scikit-learn, and a beautiful glassmorphism UI with neon highlights.
